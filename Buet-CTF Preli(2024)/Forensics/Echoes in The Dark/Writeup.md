# Binary Image Conversion from Encoded Data
## Points - 100
The challenge presented a data string in the format `width::height::data`, where:
- `width` and `height` specify the image dimensions.
- `data` contains only `!` and `?` symbols, where `!` represents `0` (black) and `?` represents `1` (white).

This format indicates that the data can be visualized as a binary image, with `0`s and `1`s representing pixel colors.

## Approach

1. **Extract Dimensions and Data**: We first split the input string to retrieve `width`, `height`, and `data`.
2. **Binary Conversion**: Each `!` was converted to `0` and each `?` to `1` to create a binary string.
3. **Reshape for Image Display**: We reshaped this binary data into a 2D array with the extracted dimensions to match the intended image layout.
4. **Display the Image**: Using `matplotlib`, we plotted the data as a black-and-white image.
